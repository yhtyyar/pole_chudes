# UI/UX & Design System Document

## 1. Цветовая палитра (Theming)
Проект использует CSS переменные для поддержки светлой (`data-theme="light"`) и темной (`data-theme="dark"`) тем.

### Базовые цвета (Темная тема по умолчанию)
- `--color-bg`: `#0f172a` (Глубокий синий фон)
- `--color-panel`: `#1e293b` (Фон панелей)
- `--color-card`: `#334155` (Фон карточек игроков)
- `--color-text`: `#f8fafc` (Основной текст)
- `--color-text-muted`: `rgba(248,250,252,0.65)` (Второстепенный текст)
- `--color-border`: `rgba(255,255,255,0.1)` (Границы)

### Базовые цвета (Светлая тема)
- `--color-bg`: `#eef2f7`
- `--color-panel`: `#dce4f0`
- `--color-card`: `#c8d4e8`
- `--color-text`: `#1a1a2e`
- `--color-text-muted`: `rgba(26,26,46,0.65)`
- `--color-border`: `rgba(26,26,46,0.12)`

### Цвета секторов барабана
Заданы как CSS переменные (напр. `--drum-points`, `--drum-bankrupt`, `--drum-prize`), чтобы адаптироваться под общую яркость темы.

## 2. Типографика
- Основной шрифт: Системный sans-serif (через Tailwind CSS).
- Размеры: от `text-xs` (12px) для подписей до `text-7xl` (72px) для карточек букв на табло.
- Использование `tabular-nums` для цифр (очки, таймер), чтобы избежать дергания ширины при смене цифр.

## 3. Иконки
Используется библиотека `lucide-react` (SVG иконки). Эмодзи полностью исключены для обеспечения кроссплатформенного единообразия.
Примеры: `Play`, `Pause`, `Skull` (для банкрота), `Trophy` (для победителя), `SlidersHorizontal` (для панели админа).

## 4. Компоненты UI (Components)

### 4.1. Табло (Board)
- **Cell (Клетка):** 
  - Динамический размер: Чем длиннее слово, тем меньше клетка (адаптивно от 88px до 52px).
  - Состояние "Скрыто": Непрозрачный блок, градиент.
  - Состояние "Открыто": 3D Flip анимация по оси Y, появление буквы с glow-эффектом (светящаяся тень).

### 4.2. Карточки игроков (Player Cards)
- Текущий игрок выделяется яркой рамкой (`ring-2 ring-blue-500`) и подсветкой фона.
- Индикация очков: Глобальный счет (крупно), очки за раунд (мелко, зеленым цветом `+XXX`).
- Статус "Банкрот": Имя перечеркивается, вместо очков иконка Черепа, карточка тускнеет (`opacity-50`).

### 4.3. Барабан (Drum)
- Рисуется через SVG + CSS conic-gradient / Canvas (в зависимости от реализации, сейчас HTML/CSS трансформации).
- Имеет мини-версию (в сайдбаре) и фуллскрин оверлей при вращении.
- Индикатор выбранного сектора (стрелка сверху).

## 5. Анимации (Framer Motion)
- **Монтирование страниц:** Fade-in / Fade-out (`AnimatePresence`).
- **Модальные окна (Round Complete):** Scale-in с эффектом пружины (`type: 'spring', bounce: 0.5`).
- **Буквы на табло:** Вращение `rotateY` от 180deg до 0deg с длительностью 0.6s.

## 6. Layout & Responsive
- **Подход:** Mobile-first, но с ориентацией на десктоп/планшет (так как это игра для компании).
- **Структура Game.tsx:**
  - `Header`: Верхняя панель (Счет, раунд, элементы управления звуком/темой).
  - `Left Panel`: Панель ведущего, список игроков (скрывается/сворачивается на узких экранах).
  - `Center`: Вопрос и Табло с буквами.
  - `Right Panel`: Барабан и Таймер (переходит вниз на мобильных устройствах).
- Точки перелома (Breakpoints): Tailwind дефолтные (`md:`, `lg:`, `xl:`). Максимальная ширина центральной области ограничена `max-w-4xl`.
