# 🎡 Поле Чудес — Корпоративная игра

Полнофункциональное SPA для проведения корпоративной игры «Поле Чудес».
**Frontend-only** — работает без сервера, данные хранятся в LocalStorage.

---

## 🚀 Быстрый старт

```bash
npm install
npm run dev
# Откройте http://localhost:5173
```

## 🏗️ Сборка для продакшена

```bash
npm run build
# Готовые файлы в папке dist/
npm run preview  # локальный просмотр
```

## 🎮 Как проводить игру

### 1. Настройка (экран Setup)
- Введите названия 5 групп
- Введите 6 слов (5 отборочных + 1 финальное) — только кириллица, 3–15 букв
- Введите вопрос-загадку к каждому слову
- Нажмите «🚀 Начать игру»
- Конфигурацию можно сохранить/загрузить через JSON

### 2. Порядок хода
1. Нажать «🎰 Крутить барабан» (или Space)
2. Ввести букву → нажать «✓ Назвать букву» (или Enter)
3. Система автоматически открывает буквы / переходит к следующему игроку

### 3. Сектора барабана

| Сектор   | Действие                          |
|----------|-----------------------------------|
| 100–1000 | Очки за каждую найденную букву    |
| ×2       | Текущие очки раунда удваиваются   |
| +1       | Дополнительная буква (ход остаётся)|
| БАНКРОТ  | Очки раунда сгорают               |
| ПРИЗ     | Особый приз (решение ведущего)    |
| БАНК     | Накопленные очки                  |

### 4. Горячие клавиши

| Клавиша | Действие                  |
|---------|---------------------------|
| Space   | Крутить барабан            |
| Enter   | Подтвердить букву          |
| Esc     | Пауза / продолжить таймер  |
| N       | Следующий игрок            |

### 5. Панель ведущего
Нажмите «🎛 Панель ведущего» для управления:
- Следующий игрок / Пропустить ход
- Принудительный банкрот / Объявить победителя
- Пауза таймера / Открыть букву вручную

## 💾 Данные и резервные копии
- Автосохранение в LocalStorage после каждого действия
- F5 — игра восстанавливается автоматически
- Кнопка «💾 Бэкап» — скачать JSON с полным состоянием

## ⚙️ Технологии
- React 19 + TypeScript 5
- Vite 8 + Tailwind CSS 3
- Zustand 5 (state management)
- Framer Motion 11 (анимации)
- canvas-confetti (финальный экран)
- Web Audio API (синтетические звуки, файлы не нужны)
