## 📖 Описание

«Поле Чудес» — это классическая телевизионная игра, адаптированная для проведения вживую на мероприятиях. Ведущий загадывает слова, участники по очереди крутят барабан и угадывают буквы. Приложение полностью берёт на себя механику: барабан, таймер, подсчёт очков, переходы между раундами и финал.

### Ключевые возможности

- **Гибкая настройка** — от 1 до 8 групп, произвольное число игроков в каждой, до 8 раундов + финал
- **Режим «только финал»** — быстрый формат без отборочных раундов
- **Интерактивный барабан** — анимация вращения с секторами очков, удвоения, банкрота и приза
- **Таймер** — 20 секунд на ход с паузой, продлением и сбросом
- **Панель ведущего** — полный контроль: смена игрока, банкрот, открытие букв, объявление победителя
- **Автосохранение** — состояние игры сохраняется в браузере, можно перезагрузить страницу
- **Тёмная и светлая темы** — переключение одной кнопкой
- **Горячие клавиши** — Space (барабан), Enter (буква), Esc (пауза), ? (правила)
- **Таблица лидеров** — очки сохраняются между играми
- **Без сервера** — 100% frontend, развёртывается как статический сайт
- **Адаптивный дизайн** — работает на Full HD проекторе и мобильных устройствах

## 🎮 Правила игры

### Цель
Угадать загаданное слово, называя буквы по одной после каждого вращения барабана. Побеждает игрок, набравший наибольшее количество очков.

### Структура
- **Группы** — от 1 до 8 команд. В каждом раунде участвует одна группа.
- **Раунды** — каждый раунд — новое слово и новая группа.
- **Финал** — необязательный финальный раунд для победителей каждой группы.

### Ход игры
1. Ведущий зачитывает вопрос-подсказку.
2. Игрок крутит барабан (кнопка или клавиша **Space**).
3. Барабан останавливается на секторе — игрок называет букву.
4. **Буква есть в слове** → она открывается, игрок получает очки и крутит снова.
5. **Буквы нет** → ход переходит к следующему игроку.
6. Игрок может угадать слово целиком — правильно = победа в раунде, неправильно = потеря хода.
7. Раунд завершается, когда слово открыто или ведущий объявляет победителя.

### Секторы барабана

| Сектор | Действие |
|--------|----------|
| **100–1000** | Очки за каждую найденную букву в слове |
| **×2 Удвоение** | Все очки текущего раунда удваиваются |
| **+1 Доп. буква** | Назовите букву — 100 очков за каждое вхождение |
| **ПРИЗ** | Специальный приз от ведущего + 100 очков за букву |
| **БАНКРОТ** | Все очки текущего раунда сгорают, ход переходит дальше |

### Очки
- **Очки раунда** — накапливаются в течение раунда (сгорают при банкроте).
- **Общий счёт** — суммируется за все раунды. Победитель раунда получает **+1500 бонусных** очков.

### Горячие клавиши

| Клавиша | Действие |
|---------|----------|
| **Space** | Крутить барабан |
| **Enter** | Подтвердить букву |
| **Esc** | Пауза / продолжить таймер |
| **?** | Открыть правила игры |

## 🚀 Быстрый старт

```bash
# Клонируйте репозиторий
git clone https://github.com/yhtyyar/pole_chudes.git
cd pole_chudes/field-of-wonders

# Установите зависимости
npm install

# Запустите в режиме разработки
npm run dev
```

Откройте **http://localhost:5173** в браузере.

## 🏗️ Сборка и деплой

```bash
# Запуск тестов + сборка
npm run build

# Просмотр собранной версии
npm run preview
```

Собранные файлы находятся в папке `dist/`. Их можно разместить на любом статическом хостинге (GitHub Pages, Netlify, Vercel и т.д.).

### GitHub Pages
Проект автоматически деплоится на GitHub Pages при пуше в `main`. CI/CD настроен через GitHub Actions (`.github/workflows/deploy.yml`).
