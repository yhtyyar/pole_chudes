# 📋 Детальный план корпоративной игры «Поле чудес»

| Параметр                               | Значение                          |
| ---------------------------------------------- | ----------------------------------------- |
| Общее количество игроков | 25 человек                         |
| Количество групп                | 5 групп                              |

| Всего игр                              | 6 (5 отборочных + 1 финал) |
| Длительность одной игры   | 5 минут                              |
| Время на ответ (буква)        | 15 секунд                           |
| Общее время мероприятия   | 25-30 минут                          |
| Язык игры                              | Русский                            |

## 2. ⏱️ Структура тайминга одной игры (5 минут)


| Этап                                  | Длительность | Описание                               |
| ----------------------------------------- | ------------------------ | ---------------------------------------------- |
| Выход игрока                   | 5 сек                 | Игрок занимает место         |
| Зачитывание вопроса     | 20 сек                | Ведущий читает загадку     |
| Обдумывание буквы         | 15 сек                | Таймер на ответ                   |
| Открытие буквы               | 5 сек                 | Помощник открывает букву |
| Сектор барабана             | 10 сек                | Кручение/выбор сектора     |
| Переход к следующему    | 5 сек                 | Смена игрока                        |
| **Среднее кол-во ходов** | **8-10 ходов**      | Зависит от слова                 |

### 📊 Расчёт времени на игру

```
Вопрос (20 сек) + [Ход (15+5+10+5 сек) × 8 ходов] + Финал (30 сек) = ~5 минут
```

## 4. ✅ Чек-лист подготовки

### 📝 Документы и материалы

```
□ 6 вопросов-загадок (5 отборочных + 1 финал)
□ 6 слов для отгадывания (проверенные)
□ Список игроков по группам (распечатанный)
□ Протокол результатов игры
□ Сценарий для ведущего
```

### 🎲 Оборудование

```
□ Барабан (физический или цифровой)
□ Табло для букв (экран/доска/проектор)
□ Таймер с звуковым сигналом (15 сек)
□ Микрофон для ведущего
□ Звуковая система
```

### 🎁 Призы

```
□ 5 призов для победителей отборочных туров
□ 1 главный приз для финалиста
□ Утешительные призы (опционально)
```

### 👥 Команда организации


| Роль           | Количество | Обязанности                                           |
| ------------------ | -------------------- | ---------------------------------------------------------------- |
| Ведущий     | 1                    | Ведение игры, зачитывание вопросов |
| Помощник 1 | 1                    | Управление табло (открытие букв)      |
| Помощник 2 | 1                    | Контроль таймера                                  |
| Помощник 3 | 1                    | Работа с барабаном                               |

## 5. 🔄 Ход игры (пошагово)

### Для каждой отборочной игры:

```
1. Вызов первого игрока группы → 5 сек
2. Зачитывание вопроса-загадки → 20 сек
3. Игрок называет букву → 5 сек
4. Таймер на обдумывание → 15 сек
5. Проверка буквы в слове → 5 сек
6. Открытие буквы на табло → 5 сек
7. Сектор барабана (очки/приз/банкрот) → 10 сек
8. Переход к следующему игроку → 5 сек
9. Повторять шаги 3-8 до отгадывания слова
10. Фиксация победителя → 10 сек
```

### Для финальной игры:

```
1. Вызов 5 победителей → 30 сек
2. Зачитывание финального вопроса → 30 сек
3. Игра по тем же правилам (15 сек на букву)
4. Определение финального победителя
5. Вручение главного приза
```

## 6. ⚙️ Правила и сектора барабана

### Стандартные сектора:


| Сектор   | Значение     | Действие                                   |
| -------------- | -------------------- | -------------------------------------------------- |
| 1-10           | Очки             | Начисление очков за букву    |
| x2             | Удвоение     | Очки удваиваются                    |
| +              | Плюс             | Право на ещё одну букву         |
| БАНКРОТ | Потеря         | Ход переходит, очки сгорают |
| ПРИЗ       | Награда       | Возможность выиграть приз   |
| БАНК       | Сохранение | Очки сохраняются в банке      |

### Правила для корпоративного формата:

```
□ Побеждает игрок с наибольшим количеством очков
□ При банкротстве очки текущего раунда сгорают
□ При отгадывании слова — автоматическая победа
□ В финале — игра до отгадывания слова
```

## 7. ⚠️ Риски и решения


| Риск                              | Вероятность | Решение                                             |
| ------------------------------------- | ---------------------- | ---------------------------------------------------------- |
| Превышение времени   | Средняя         | Строгий таймер, сигнал                  |
| Спорная буква             | Средняя         | Решение ведущего окончательно   |
| Технический сбой       | Низкая           | Бумажная версия табло в резерве |
| Неравная активность | Высокая         | Чёткая очередность ходов             |
| Слово не отгадано      | Средняя         | Ведущий открывает подсказку       |
